# Palette's Journal - Android UX Learnings

- Always ensure `contentDescription` is present and meaningful for interactive icons.
- Use Haptic feedback (`HapticFeedbackType.LongPress` or `vibrate`) for critical primary actions.
- Use Material 3 ColorScheme and Typography.
- Touch targets should be at least 48dp.
- Smooth state transitions with `AnimatedVisibility` or `Crossfade`.
